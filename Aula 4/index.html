O que é o HTML?

O HTML é uma linguagem de marcação usada para estruturar páginas na web. Ele define títulos, parágrafos, imagens, links, etc.

O que é o CSS?

O CSS é uma linguagem usada para estilizar páginas HTML. Ele controla cores, espaçamentos e os layout dos elementos.
